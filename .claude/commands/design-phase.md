# /design-phase

> Execute Design phase activities for the AI Capstone - solution architecture, feature specification, and PRD development.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@capstone-agent`
2. **Skill instructions**: `skills/capstone/design-phase.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Helps with Design phase activities:
- Solution ideation and AI approach selection
- Product architecture design
- Feature specification and prioritization
- User experience design
- PRD development

## Quick Use

Run `/design-phase` and specify what you need:

| Action | Description |
|--------|-------------|
| `status` | Current Design phase and PRD progress |
| `solution-ideation` | Generate and evaluate solution concepts |
| `architecture` | Design system architecture |
| `features` | Specify and prioritize MVP features |
| `user-flows` | Create user journey flows |
| `prd` | Work on PRD sections |
| `prd-review` | Review and improve PRD draft |

## Usage Examples

```
/design-phase status
```
Shows Design phase progress and PRD completion status.

```
/design-phase solution-ideation
```
Generate and evaluate AI/AGI solution concepts.

```
/design-phase architecture
```
Design the product architecture with AI components.

```
/design-phase prd
```
Work on PRD sections (will ask which section).

```
/design-phase user-flows
```
Create user flows for core journeys.

## PRD Sections

9. Risks & Mitigations
10. Timeline
