# /deploy-phase

> Execute Deploy phase activities for the AI Capstone - launch planning, success metrics, and final presentation.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@capstone-agent`
2. **Skill instructions**: `skills/capstone/deploy-phase.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Helps with Deploy phase activities:
- Go-to-market planning
- Success metrics definition
- Presentation preparation
- Final deliverables compilation
- Capstone retrospective

## Quick Use

Run `/deploy-phase` and specify what you need:

| Action | Description |
|--------|-------------|
| `status` | Current Deploy phase and submission progress |
| `gtm` | Work on go-to-market planning |
| `metrics` | Define success metrics |
| `presentation` | Work on capstone presentation |
| `demo-polish` | Polish the product demo |
| `deliverables` | Check all deliverables status |
| `retrospective` | Create capstone retrospective |

## Presentation Structure

1. Hook (1-2 min)
2. Problem & Opportunity (3-4 min)
3. Solution + Demo (4-5 min)
4. Technical Approach (2-3 min)
5. Validation & Results (2-3 min)
6. Go-to-Market (2 min)
7. Journey & Learnings (2 min)
8. Future Vision (1-2 min)

**Total**: ~20 minutes

## Exit Criteria

- [ ] Go-to-market plan complete
- [ ] Success metrics defined
- [ ] Presentation ready and rehearsed
- [ ] Final demo polished
- [ ] Capstone submitted
